Basic (pre-2016) state pension is now taken directly from reported FRS amounts and is no longer parametrised — entitlements depend on contribution histories the model cannot reconstruct. The `old_basic_pension_weekly` parameter and the baseline-rate plumbing threaded through every constructor are removed; only `new_state_pension_weekly` remains reformable. Zero-reporting pensioners above the basic SP age no longer get an imputed full-rate award, reducing baseline state pension spending by £0.86bn (0.6%) on 2025/26 FRS data.
