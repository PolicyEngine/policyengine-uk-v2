Extend the PIP-from-flags pattern to DLA care/mobility components and Attendance Allowance. New `DlaParams` and `AaParams` structs (with 2025/26 weekly rates from gov.uk under SSCBA 1992 Sch.2 paras 2–3 and SSCBA 1992 s.64) are exposed in the Python wrapper. Synthetic households built via `Simulation.from_situation` that set the `dla_care_*` / `dla_mob_*` / `aa_*` flags now produce non-zero amounts; FRS-recorded amounts continue to pass through unchanged.
